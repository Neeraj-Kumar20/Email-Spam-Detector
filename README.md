# Email-Spam-Detector
"A Machine Learning web application built with Streamlit and Python to detect whether an email is Spam or Ham using the Multinomial Naive Bayes algorithm."
# 🛡️ Email Spam or Not Spam Detector

This is a **Machine Learning** project developed as part of the **MSc IT Final Year Project**. The application uses **Natural Language Processing (NLP)** and the **Naive Bayes** algorithm to classify emails as either 'Spam' or 'Ham' (Not Spam).

## 🚀 Features
- **User-Friendly Interface**: Built with a custom orange-pink gradient theme using **Streamlit**.
- **Real-time Prediction**: Instantly analyzes the entered text.
- **Data Preprocessing**: Includes text cleaning and TF-IDF vectorization for high accuracy.

## 🛠️ Technology Stack
- **Language**: Python
- **Framework**: Streamlit
- **Libraries**: Pandas, Scikit-learn, Pickle, NLTK
- **Algorithm**: Multinomial Naive Bayes
